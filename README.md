# Data-Structures-
A module which covers the data structures from Basic to Advanced. This is divided into 3 parts Basic, Advanced and Advanced 2.0

Basic DSA:
    1.) Arrays
    2.) Stacks and Queue
    3.) Hashing
    4.) Heap (Priority Queue)
    5.) 2 Pointers
    6.) Window Sliding
    7.) Linked List
    
Advanced DSA:
  1.) Recursion
  2.) Dynamic Programming
  3.) Backtracking
  4.) Graphs
  5.) Trees
  6.) BST
  7.) Bit Manipulation and Bitmasking

Advanced 2.0 DSA:
  1.) String Algo
  2.) Binary Lifting
  3.) Segment Tree
  4.) Tries
  5.) Fenwik Trees
  6.) Range Querries
